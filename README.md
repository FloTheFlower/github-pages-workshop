# Build Your Own Portfolio and Blog Site with GitHub Pages Workshop

```
$ git clone https://github.com/SuzeShardlow/github-pages-workshop.git
$ cd github-pages-workshop
$ bundle install
$ bundle exec jekyll serve --watch
```
